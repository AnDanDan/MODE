Tool selected: MIT App Inventor

Platform: Android
Native App development
Native App Advantages: Better to make user of a specific OS features, easier time bug testing
Native App Disadvantages: Limited to that OS
Website: MIT App Inventor
Developers: Originally Google, now MIT
Initial Release: December 15, 2010
Liscence: MIT Liscence
Price: Free
Languages: GUI, underlying code is Java based.
Features: GUI interface for drag and drop creation
Debugger: None
Built In Emulator: Yes
Host Machine Requirements: Basic. Enough to run a web browser to access the website.
IDE: None. It is a web service.
Software to Install: None. Will run in your OS' native web browser
Advantages: Easy to learn, users don't need to learn complex coding languages to make apps
Disadvantages: Extremely limited scope in what can be done. Android only.
Language Availibility: English, French, Spanish, Italian, Korean, Dutch, Porteguese, Brazilian Porteguese, Russian, Swedish, Chinese (Traditional), Chinese (Simplified)
